# Практична робота 9

## Тема
Основи роботи та комунікації з React Router.

## Завдання
Зробити рефакторинг коду попередньої роботи з додаванням React Router.

## Реалізовано
- Express-сервер з REST API.
- React-клієнт.
- React Router для маршрутів `/`, `/user`, `/admin`, `/super-admin`, `/dashboard/*`.
- Proxy між React і Express.
- Отримання користувачів через API.
- Додавання попереджень адміністратором.
- Зміна ролей супер адміністратором.

## Запуск backend
```bash
cd server
npm install
npm start
```
Backend запускається на `http://localhost:5000`.

## Запуск frontend
```bash
cd client
npm install
npm start
```
Frontend запускається на `http://localhost:3000`.
